PRINTING AND COMMENTS

To print on LUA is pretty easy, you just write "print("text you want to print")" 
and would be it, there's also multiple ways of printing if you don't want to use 
the double quotes (I will put a picture of them here), also, you can see the 
things you print on the console!

Comments are with double dash (--)

DATA TYPES

There's 4 data types, strings, numbers, boolean and nil:

- Strings stands for text (has to be always surrounded by double quotes)
- Numbers, well, are numbers, can be decimal or full numbers
- Booleans are true or false
- Nil is nothing (literally, nothing)

VARIABLES

Variables are containers that holds values, for example, we have the variable var1,
and that variable is equal to "Hello World!", then if we print that variable we'll
get "Hello World!" as a result!
